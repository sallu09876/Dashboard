Ecom Clothing Dashboard (React + Vite)

How to run:
1. unzip or extract files
2. cd ecom-dashboard-vite
3. npm install
4. npm run dev

Notes:
- This is a starter scaffold using React + Vite (JavaScript).
- Uses Chart.js + react-chartjs-2 and framer-motion for animations.
- Styling is plain CSS (src/styles.css) with a neon premium look.
- Sidebar switches pages; charts are placeholders with sample data.

Customize as you like. Enjoy!
